# **App Name**: EnviroWise EIA

## Core Features:

- Project Dashboard: Display project description (name, description, date, authors).
- Alternative Listing: Show a list of proposed project alternatives.
- Factor Management: Enable users to define and manage environmental factors and their weights.
- Alternative Creation: Create new alternatives, input a name, then populate the alternative.
- Effect Definition: Define actions, environmental factors, and resulting effects.
- Fuzzy Logic Assistant: AI-powered tool that guides users in assigning a character (compatible, critical, moderate, severe) to the environmental impact by presenting questions about the nature of the environmental effects and adjusting the idoneity using a slider; after collecting the relevant data points, the LLM will 'Infer' the character of the effect using fuzzy logic.
- Results Visualization: Display final project results and comparison of valued alternatives.

## Style Guidelines:

- Primary color: Saturated teal (#4DB6AC) to reflect environmental awareness and trustworthiness.
- Background color: Light teal (#E0F2F1), subtly desaturated, provides a calm and professional backdrop.
- Accent color: A vivid lime green (#AEEA00), to highlight interactive elements and indicate a clear path of navigation.
- Body font: 'Inter' sans-serif for clear and neutral readability of project information.
- Headline font: 'Space Grotesk' sans-serif for a computerized feel.
- Use environmentally-themed icons to visually represent factors and actions.
- Subtle transitions and animations when navigating between alternatives and effects.
